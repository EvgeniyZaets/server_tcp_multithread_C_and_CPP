Подробное описание

Структура посылаемого сообщения.

Эта структура содержит указатель на сохраненное сообщение (пакет) и идентификатор таймера. Поле pkt является указателем на сохраненное сообщение типа packet_t. Поле id представляет собой идентификатор таймера, который используется для ретрансмиссии. Поле start представляет собой структуру timeval для контроля времени.
