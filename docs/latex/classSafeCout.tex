\doxysubsection{Подробное описание}
Класс \mbox{\hyperlink{classSafeCout}{Safe\+Cout}} обеспечивает безопасный вывод в стандартный поток вывода (cout) с использованием мьютекса.

Класс \mbox{\hyperlink{classSafeCout}{Safe\+Cout}} предоставляет функциональность для безопасного вывода данных в стандартный поток вывода (cout) с использованием мьютекса. Он перегружает оператор \char`\"{}$<$$<$\char`\"{} для различных типов данных, а также для строк и манипуляторов потока. При использовании оператора \char`\"{}$<$$<$\char`\"{} объекта \mbox{\hyperlink{classSafeCout}{Safe\+Cout}}, мьютекс блокируется, чтобы гарантировать безопасность вывода в многопоточной среде.

Пример использования\+: 
\begin{DoxyCode}{0}

\DoxyCodeLine{safeCout << \textcolor{stringliteral}{"{}Hello, world!"{}} << std::endl;}

\end{DoxyCode}


В этом примере создается объект класса \mbox{\hyperlink{classSafeCout}{Safe\+Cout}} и используется оператор \char`\"{}$<$$<$\char`\"{} для безопасного вывода строки \char`\"{}\+Hello, world!\char`\"{} и символа новой строки. Мьютекс блокируется во время вывода, чтобы гарантировать безопасность вывода в многопоточной среде.
