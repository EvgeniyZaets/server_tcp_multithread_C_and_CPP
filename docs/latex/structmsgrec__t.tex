\doxysubsection{Подробное описание}
Структура посылаемого сообщения. 

Эта структура содержит указатель на сохраненное сообщение (пакет) и идентификатор таймера. Поле {\ttfamily pkt} является указателем на сохраненное сообщение типа {\ttfamily \mbox{\hyperlink{structpacket__t}{packet\+\_\+t}}}. Поле {\ttfamily id} представляет собой идентификатор таймера, который используется для ретрансмиссии. Поле {\ttfamily start} представляет собой структуру {\ttfamily timeval} для контроля времени.
