\doxysubsection{Подробное описание}
Структура пакета сообщения. 

Эта структура представляет собой пакет сообщения, содержащий длину признака и данных, признак сообщения и само сообщение. Поле {\ttfamily len} используется для указания длины признака и данных, и удаленный хост может использовать это поле для разбиения на отдельные записи. Поле {\ttfamily cookie} является 32-\/битным порядковым номером сообщения. Поле {\ttfamily buf} представляет собой буфер для хранения сообщения.
